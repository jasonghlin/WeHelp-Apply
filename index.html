個人簡介

老師您好，我叫林干紘，大學畢業於臺北醫學大學藥學系，畢業後繼續攻讀研究所兩年並任職於台大醫院的藥劑部。在就讀研究所期間因為碩論需要，曾利用 R 語言做統計分析，那時就覺得寫 Code、看到 Code 能動是一件好玩、有成就感的事情。而在醫院工作的期間，因為工作性質的關係，每天除了必須要像打仗一樣整天對任何事情做條件反射，還要接觸各式各樣的人並和他們溝通。我發現這樣的工作型態不是我喜歡的，除了每天做很多重複性的工作之外，面對病人很耗費心力但獲得的成就感卻不多，也因此萌生想轉職的念頭。而剛好那時候在醫院也待了一段時間，各部門的受訓告一個段落，想趁著還年輕希望可以看看更多不一樣的世界，因而離職開始思考轉職的事情。會希望可以當軟體工程師的原因是彈性很高，而且工作的選擇相對於藥師而言也比較多，更重要的是在過去的經驗中，對於可以親自寫 Code 讓程式動起來幫忙完成工作這件事讓我覺得很開心，這也是讓我可以繼續學習程式語言的動力，雖然過程中會有很多次的失敗，但每次的成功都代表著解決了一個問題，也同時帶來成就感。

為了成為軟體工程師，做過什麼努力？

在離職過後我先參加了政府舉辦的 T 大使計劃，希望可以藉由計畫了解一下如何利用程式或是數位工具解決問題。在這之後為了持續的精進，我一開始是先去購買書籍並嘗試照著書中的範例做練習，利用這樣的方式來再次確認自己是不是真的喜歡寫程式。之後為了能夠加快這個進程以及讓自己增加多一點的成就感我做了幾件事，首先是報名台大的資訊系統訓練班課程，藉由老師的教學來縮短學習當中可能會遇到的問題以及時間。另一方面我也去上了 Udemy 的線上課程以及觀看 YT 上的免費資源，藉由實際的範例操作來加強自己的成就感與信心。

如果參與這個訓練，會怎麼安排學習時間？

如果可以順利錄取，我會保持是全職學習的狀態，平日除了吃飯、睡覺與運動的時間之外，會將時間盡量按排在學習，因此會大概有 9-11小時都在寫程式。假日則會安排 6-8 小時的學習，剩下的時間讓自己稍微休息一下思考這週的進度以及規劃隔週的進度。

是否有想要加入的公司？為什麼想加入該公司？

我希望可以加入醫療軟體相關的公司，原因是希望可以結合過去的經驗，除了可能可以減少過程中的適應時間之外，可以結合過去的興趣也是我期望的目標，尤其是如果真的可以改變一些現況會更有成就感。

請描述一件產生明顯負面情緒的經歷，如何處理該情緒？

在這之前也有申請過其他的 Bootcamp，那個時候收到沒有錄取時心情滿沮喪的，除了沒被錄取讓我感到情緒低落之外，最讓我感到迷惘的是我沒有得到任何的 Feedback，不知道該往哪裡做加強才是各個 Bootcamp 所希望具備的能力。我解決情緒的方式是先讓當下沮喪的情緒得到抒發，找個朋友聊天，並持續進行運動。最重要的是找到這個問題可能的解決方案：去上課。透過上課的方式來精進自己之外，也可以透過這個方式來推測可能需要具備的能力。我是藉由這些方式來處理當時的情緒。

關於這份申請網頁，分享一個開發時的技術心得。

在規劃網頁的時候，因為背景是空白的關係，想加入一個開關燈的按鈕讓背景可以按照習慣做 Dark mode 的切換。本來以為要用到非常複雜的 JavaScript 撰寫，後來發現可以利用 CSS 的方式做出開關時候有點像是看到新的世界，我利用 Checkbox 來模擬點擊的 Event，並利用 Label 與 Checkbox 關聯的特性把 Label 的樣式 做成開關的圖案，再利用選擇器做按鈕位置的位移，最後再搭配 JavaScript DOM 做樣式的切換來完成整個 Dark mode 按鈕的開發。完成這個按鈕的感想是， CSS 完全就是想像力的世界，藉由各種 CSS 的樣式可製作出很多精美的排版與初步的樣式甚至到特效，而 JavaScript DOM 的加入讓整個網頁的開發可以做到更多複雜的事，也開始慢慢體會到為什麼可以看到這麼多漂亮網頁的實質原因。
